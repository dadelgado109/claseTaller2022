Comandos GIT

Listado comandos GIT


Comando Descripción Ejemplo
init Este comando se utiliza para crear un repositorio git 
git pull origin main
clone Clonar un repositorio local o remoto 
git clone https://git...../proyecto.git myProyecto
pull Commado que se emplea para descargar contenido desde el repositorio remoto y actualiza al instante el repositorio local 
git pull origin main
branch Este comando se utiliza para trabajar con todas las ramas que hay en nuestro entorno de desarrollo 
  • git branch (Muestras todas las ramas)
  • git branch <miRama> (Crea una nueva rama llamada miRama)
  • git branch -d <miRama> (Elimina la rama miRama)
  • git branch -D <miRama> (Elimina la rama miRama de forma forzada)
  • git branch -m <miRama> (Cambia el nombre de la rama actual)
  • git branch -a <miRama> (Enumera las ramas remotas eso incluye ramas de compañeros)
status Nos muestras los archivos que fuimos modificando en nuestra rama 
git status
stash Este comando saca una foto de nuestros cambio y deja el desarrollo con los cambios de nuestro último pull 
  • git stash (Saca la foto y restablece el código)
  • git stash pop (Recuperamos la foto)
add Este comando agrega archivos para subir subir al repositorio 
  • git add <ruta archivo especifica> (Agrego un archivo en especifico)
  • git add * (Agrego todos los archivos modificados)
commit Este comando sirve para marcar que un trabajo esta completado en el repositorio local. 
  • git commit
  • git commit -m "Comentario" (Agrego un comentario)
checkout Este comando te permite cambiar de una rama a otra rama 
  • git checkout <nombreRama> (Me muevo a la rama nombreRama)
  • git checkout -b <nuevaRama> (Creo una rama nuevaRama y me muevo a ella)
merge Se utiliza para unificar 2 ramas 
git marge <ramaExitente>
push Este comando sube al repositorio el cambio guardado con el commit. 
  • git push (Sube todo los cambios a la rama principal)
  • git push <rama> (De esta forma se suben los cambios a esta rama)
  • git push <rama> --force (Forzamos la suba de cambios a la rama)
